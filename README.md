# Purple Fire Challenge

This project is part of a challenge to evaluate the frontend skills. This project was created with NextJS.

## Challenge overview

The goal of this challenge is to create a responsive product detail page.


## Prerequisites

To be able to install the project it is needed the following:

- node (v >= 20.8.0)

## Technologies used 

The technologies used to develop this project are the following:

- NextJS (>= v 14.1.4)
- React (>= v 18)
- @mui/material (>= v 5.15.15)

## Getting started

Follow these steps to set up the project locally:
1. Clone de repository from https://github.com/DanielLorenzoCabrera/purpleFireChallenge.git
2. Navigate to the project directory: ***"cd purple-fire-challenge"***
3. Install the dependencies: ***npm install***

## Usage

1. Run the application in development mode: ***npm run dev***
2. Open your browser at **localhost:3000**
